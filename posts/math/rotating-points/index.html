Rotating Points

When playing with 2d/3d computer graphics, image processing, computer vision,etc we often need use the rotation matrix to rotate a point <x,y> or <x,y,z> around origin or around an arbitrary point.

For example, on the following animation, we need to compute the new x’ and y’ of the endpoint of the blue vector. On every frame, we increase the angle by 0.05 rads and compute the new x,y values.

To do that, we use the standard point rotation formula, which is commonly stated as that if you have a point (x, y) and you want rotate by \( \theta \).

The new x and y values are given by

$$ x’ = x \cos \theta - y \sin \theta \newline y’ = y \cos \theta + x \sin \theta $$

Example#

For example, if we have a point at (4,3) and want to rotate by 30 deg, the new point x’ and y’ is

x’ = 4 cos30 - 3sin30

y’ = 3 cos30 - 4sin30

Using cos(30deg) = 0.866 and sin(30deg) is 0.5

x’ = 4 * 0.866 - 3 * 0.5

x’ = 0.616 + 2.964

x’ = 3.58

Point Rotation

When playing with 2d/3d computer graphics, image processing, computer vision, etc., we often need use the rotation matrix to rotate a point <x,y> or <x,y,z> around origin or around an arbitrary point.

For example, the following animation, we rotate the red point by 15 degree every half second. That means that we need compute the new x,y positiong from the previous x,y rotated by 15 deg.

Animation: Point rotation on canvas coord system.

To do that, we use the standard point rotation formula, which states that if you have a point defined by the values x and y on the cartesian coordinate system, and you want rotate by \( \theta \) deg couterclockwise, then the new x’ and y’ values are given by:

$$ x’ = x \cos \theta - y \sin \theta \newline y’ = x \sin \theta + y \cos \theta $$

Using matrices, we can write as

$$ \begin{bmatrix} x’ \newline y’ \end{bmatrix} \mathrel{=} \begin{bmatrix} \cos \theta & -\sin \theta \newline \sin \theta & \cos \theta \end{bmatrix} \begin{bmatrix} x \newline y \end{bmatrix} \mathrel{=} \begin{bmatrix} x \cos \theta - y \sin \theta \newline x \sin \theta + y \cos \theta \end{bmatrix} $$

For example, if we have a point at <5,0> and want to rotate by 30 deg, the new point x’ and y’ is the following. Using cos(30deg) = 0.866 and sin(30deg) = 0.5.

$$ x’ = 5 \times \cos 30^\circ - 0 \times \sin 30^\circ = 5 \times 0.866 - 0 \times 0.5 = 4.33 $$ $$ y’ = 5 \times \sin 30^\circ + 0 \times \cos 30^\circ = 5 \times 0.5 + 0 \times 0.866 = 2.5 $$

The figure below illustrate this:

Rotation Direction#

Note that the result is +2.5, not -2.5, which agrees with the statement that it is a counterclockwise rotation. But if you look at the previous animation, you will see a clockwise rotation. They both are using the same math equation. The only difference is that the canvas coordinate system has a flipped y when compared with the cartesian coordinate system. The y-axis points downwards (increasing y goes down the screen). This means on canvas, a positive +2.5 goes down. And on the cartesian, a positive +2.5 goes up.

If you render the dot with a flipped y value, from <x,y> to <x,-y>, ie, if you compute 2.5, you use -2.5, you get a anticlosewise rotation like you see on the cartesian coordinate system:

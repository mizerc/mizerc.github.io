Comment the entire line:

CMD + /

SHIFT + OPT + UP/DOWN

Increase cursor to 2 lines:

CMD + ALT + UP/DOWN

Copy entire line up or down:

OPT + SHIFT + UP/DOWN

Move entire line up or down:

OPT + UP/DOWN

Select current line:

CMD + L

Select all matches of current selected word:

CMD + SHIFT + L

CMD+K CMD+0

CMD + D

CTRL -

Regex Search#

Find word1 AND word2 on the same row, any order:

(?=.*word1)(?=.*word2)

Find word that is not followed by any other letter:

myword[^a-zA-Z]

Find any word between react-use/lib/ and drop:

react-use/lib/\\w*drop

Filter out test files:

.spec.ts,*.spec.tsx

Using folder curinga in the command pallete:

src/modules/**/selectors/

Git#

Revert all the changes of a specific commit hash:

git revert 1b4adb2

Checkout into a specific commit hash:

git checkout 12f14c7

Checkout/reset a specific file to match the file from origin/remote:

git checkout origin/main ./src/tasks/file.ts
git checkout origin/master ./src/modules/file.ts

Empty commit to trigger CI/CD:

git commit --allow-empty -m "Trigger build"

Put git logs to file:

git shortlog > gitlogs.txt

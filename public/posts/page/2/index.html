Web Audio API

Web Audio API Modern web browsers support the Web Audio API, which allows us to create and manipulate audio streams. Audio Context The AudioContext object is the main entry point for the Web Audio API. Audio Nodes and Chains The AudioNode interface is the base interface for all nodes in the audio processing graph. We can connect audio nodes together to create a chain of nodes.

The basic idea to create a fractal is to use a formula to check how stable the computation is.
If after a certain number of iterations the computation is still stable (e.g. the absolute value of the complex number is less than a certain threshold), we can say that the point is in the fractal. 
Otherwise, the point is not in the fractal.
We can define a color for each point based on the number of iterations it took to become unstable, for example, linear map the number of iteration to HUE domain (0-255)in the HSV color space.

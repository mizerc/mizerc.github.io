For visual folks that also like math, fractals can feel like the cherry of the cake that nature has given us.

The basic idea to create a fractal is to use a formula to check how stable the computation is. 

If after a certain number of iterations the computation is still stable, we can say that the point is in the fractal. Otherwise, the point is not in the fractal.

We can define a color for each point based on the number of iterations it took to become unstable.
